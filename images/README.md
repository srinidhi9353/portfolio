# Images Directory

Place your images in this folder for them to display correctly on your portfolio.

## Required Images:
- `profile.jpg` - Your profile photo (recommended size: 400x400 pixels)
- `Datastructures.png` - Data Structures certificate
- `Python.png` - Python certificate

## For GitHub Pages Hosting:

1. Add all required images to this folder
2. Commit and push to your GitHub repository
3. Enable GitHub Pages in your repository settings
4. Your portfolio will be available at https://[your-username].github.io/[repository-name]/
